# 🎅 X-mas Run 
> **[정글 9기 나만의 무기 만들기 프로젝트](https://github.com/Crazy-Cow)**  
> 웹에서 즐기는 크리스마스 선물 쟁탈 파티 게임

## ✨ Key Features
- 클릭 2번으로 가능한 간편한 시작
- 3개의 캐릭터와 캐릭터별 특수능력
- 4개 아이템 (번개, 선물, 쉴드, 부스트)
- 로그를 통한 랭킹
